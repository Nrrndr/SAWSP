cara penggunaan 
1 siapkan data dalam file 
 a file akan mulai dieksekusi dengan satu baris yg hanya terisi start  
 b setelah start tulis boolean jika true akan mengeksekusi SAW jika false SP3
 c masukan weight variable dipisah oleh spasi misal 20 20 15 10 10 15 10
 d 7 variable tersebut adalah
     profit_margin;+
    demand;+
    shelf_life;+
    retention;+
    cost;-
    competitor;-
    return_rate;-
 e tulis data dengan 
    nama profit_margin demand shelf_life retention cost competitor return_rate
    format
    String int int int int int int int
    gunakan spasi untuk pemisah
2 run dengan variable filename yang diinginkan atau 'data.txt' jika kosong
output : untuk SAW 
adalah matrix SAW untuk setiap item dan value setiap variable + hasil perhitingan saw 
dan item terbaik dan nilainya
    SP3
adalah nama item dan hasil dari perhitungan SP3 dan nama dan nilai dari item terbaik
